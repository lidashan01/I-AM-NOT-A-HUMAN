Save/Softlock — Current Status

Recent hotfixes targeted edge cases around saving at the start of the night. If you still encounter issues, verify files and avoid saving during transitions. Report with steps on the Community hub.

Controller Prompts and Input Tips

Prompt icon glitches may occur with certain controllers. Try disabling Steam Input and avoid switching between controller and keyboard in a single session.
